# NovaPay Zero-Downtime DevSecOps CI/CD Pipeline

## Overview

NovaPay Zero-Downtime DevSecOps CI/CD Pipeline is a production-grade architecture designed for a digital banking platform. The solution focuses on secure, compliant, automated, and zero-downtime software delivery while aligning with RBI technology governance expectations and PCI-DSS security requirements.

The project demonstrates how modern DevSecOps practices can be applied to financial services environments through Infrastructure as Code, automated security validation, deployment strategies, observability, and incident management.

---

## Key Features

* Zero-Downtime Deployments
* Blue-Green Deployment Strategy
* Canary Deployment Strategy
* Automated Rollback Mechanisms
* GitHub Actions CI/CD Pipeline
* Infrastructure as Code using Terraform
* Kubernetes Deployment Architecture
* RBI Compliance Controls
* PCI-DSS Compliance Mapping
* Security Gates (SAST, DAST, Dependency Scanning)
* DORA Metrics Framework
* Incident Response Playbooks
* Deployment and Rollback Runbooks
